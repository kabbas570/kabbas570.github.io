## Abstract 
Based on microphone sensor data, this IoS application classifies singing vs. speaking vs. silence of the last 2 seconds. It utilizes a deep learning-based CNN model.

A more advanced version of this project aims to use Mobile Deep Learning for Plant/ Weed Recognition and their diseases in  remote areas.
